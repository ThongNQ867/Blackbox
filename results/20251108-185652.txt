*Definition and Context:*

*Analogy/Example:* Imagine you want to build a ramp. The length of the ramp is 'c', the height it goes up is 'b', and the distance it covers horizontally is 'a'. The Pythagorean theorem helps you figure out the relationship between these lengths.

* **Section 1: Programming Languages**
*Definition and Context:* The board shows a comparison between Python and C++:
*Python -> Interpreter:* Python is generally referred to as an interpreted language. This means that the code is executed line by line by an interpreter, which translates and runs the code directly, without requiring a separate compilation step.
*C++ -> Compiler:* C++ is a compiled language. This means that the source code is first translated into machine code (or an intermediate language) by a compiler before it can be executed. The resulting executable file can then be run independently.
